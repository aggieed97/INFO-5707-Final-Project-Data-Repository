# INFO 5707 Final Project Repository

The data files in this repo were retrieved from the hoopR and wehoopR libraries. These files were retrieved for the 2024-2025 season and the 2025-2026 season through November 20, 2025.

The teams and conferences data files were retrieved from ESPN using the ESPN API.

The files are placed in this public repository to be retrieved through a Google Colab Jupyter Notebook without the dependency of downloading files.
